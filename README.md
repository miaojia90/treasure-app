## 技术栈：
react + react-router + redux  + sass + ES6/7 + webpack + fetch

## 安装依赖包
- npm install

## 开发环境
- npm run hot
-  访问 http://localhost:8090

## 生成环境
 npm run dist （发布生产版本，对代码进行混淆压缩，提取公共代码，分离css文件）

## 要点

- 简单的react、redux 的入门，
- 主要分析项目目录的结构划分规化，
- 结合工作实际项目的情况进行模块划分
- Redux 数据的初始化、更新、添加、删除、异步加载等操作
